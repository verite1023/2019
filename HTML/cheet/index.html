Sublime Text

  • 중복 선택 : Ctrl + D
  • 전체 선택 : Alt + F3
  • 다중선택 : Ctrl + Alt + up,down
  • 마우스 다중 선택 : Ctrl + 마우스선택
  • 단어 단위 이동 : Ctrl + left / right
  • 단어 단위 드래그 : Ctrl + Shift + left / right
  • 한 줄 복사 : Ctrl + Shift + D
  • 한 줄 삭제 : Shift + Delete
  • 줄 단위 이동 : Ctrl + Shift + Up / Down
  • 대문자로 : Ctrl + k + U
  • 소문자로 : Ctrl + k + L
  • 전체선택 : Ctrl + A
  • 라인선택 : Ctrl + Shift + L
  • 파일 찾기 : Ctrl + P
  • 라인으로 이동 : Ctrl + G
  • 접기 : Ctrl + Shift + [
  • 펼치기 : Ctrl + Shift + ]
  • 검색 : Ctrl + F
  • 전체 검색 : Ctrl + Shift + F
  • 이전탭 / 다음탭 : Ctrl + PageUp/PageDown
  • 끝으로 이동 : end
  • 처음으로 이동 : home
  • new process : Ctrl + Shift + N
  • new file : Ctrl + N
  • 닫기 : Ctrl + W
  • 화면 분할 : alt + shift + 1 ~ 9
  • last tab open : ctrl + shift + t
  • 현 위치 enter : ctrl + enter

Photoshop

파일(File)관련 단축키

  • 새로 만들기 : Ctrl + N
  • 열기 : Ctrl + O
  • 닫기 : Ctrl + W
  • 모두 닫기 : Alt + Ctrl + W
  • 저장 : Ctrl + S
  • 웹용으로 저장 : Ctrl + Alt + Shift + S
  • 초기상태로 되돌리기 : F12
  • 환경설정 : Ctrl + K

편집(Edit)관련 단축키

  • 실행 취소(작업 한 단계 전으로) : Ctrl + Z
  • 이전 단계(작업 한 단계 뒤으로) : Ctrl + Alt + Z
  • 다음 단계(작업 한 단계 앞으로) : Ctrl + Shift +Z

이미지(Image)관련 단축키

  • Hue/Saturation : Ctrl + U
  • 색상 균형 : Ctrl + B
  • 흑백 : Alt + Shift + Ctrl + B
  • 레벨 : Ctrl + L
  • 전경색 칠 : Alt + Del
  • 배경색 칠 : Ctrl + Del
  • 자유변형 : Ctrl + T
  • 선택 반전 : Ctrl + I
  • 이미지 투명도 조절 : 0 ~ 1

레이어(Layer)관련 단축키

  • 새 레이어 만들기 : Shift + Ctrl + N
  • 레이어 그룹화 : Ctrl + G
  • 레이어 복사 : Ctrl + J
